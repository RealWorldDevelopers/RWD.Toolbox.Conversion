RWD.Toolbox.Conversion
Welcome to the Conversion Library

This is a handy library to have. It converts Metric Measurement values and converts them to U.S. Measurement values and vise versa.

Getting Started

The following highlights are included in this project:

  • Temperature.cs - A Series of Methods used to Convert Measurements. Temperature measurements are handled within this class.

  • Volume.cs - A Series of Methods used to Convert Measurements. Volume measurements are handled within this class.

  • Mass.cs - A Series of Methods used to Convert Measurements. Mass measurements are handled within this class.
